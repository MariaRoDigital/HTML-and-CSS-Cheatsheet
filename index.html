HTML Crucials (code+meaning+example)

Code Meaning Example
<!Doctype> Including the DOCTYPE in a document ensures that the browser makes it's best attempt at following the relevant specifications in the document type. This must be the first thing in your document <!Doctype>
<html> Tells the browser that your document is an html document <!Doctype html>
<head> Here is where you link your meta info, stylesheets, and include the name that users will see on the tab to your page. If there is an external link you need to use in your code, you can include it here. !Caution: your head can get crowded when linking many font types, consider adding them as a @fontface in CSS <head><link href="resources/style.css type="text/css" rel="stylesheet">
